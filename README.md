TG Views Predict

**Решение для прогноза просмотров Telegram-рекламы.**  
Этот репозиторий содержит прагматичный и эффективный код для хакатона. Модель использует минималистичный подход с медианами по CPM и дню недели + fallback.

## Цели проекта
- Максимизировать обобщение на новых каналах (тест 19 января).
- Минимизировать зависимость от сложных библиотек (только pandas).
- Обеспечить 100% проход формата сабмита.
- Достичь конкурентного скора без переобучения.

## Структура репозитория
- `AllData.csv` — обучающая выборка (не заливаю в репозиторий по правилам хакатона).
- `TestDataset.csv` — тестовый шаблон.
- `train.py` — основной скрипт для генерации submission.
- `submission_minimal.csv` — пример сабмита (твой текущий результат).
- `submission_improved.csv` — улучшенная версия (если применишь мои правки).

## Как запустить (прагматично и быстро)
1. Установи pandas: `pip install pandas`.
2. Положи AllData.csv и TestDataset.csv в папку.
3. Запусти: `python train.py`.
4. Получишь submission_minimal.csv — заливай в систему.

## Текущий результат
- Формат — 100% проход.
- Качество — базовое (медиана по CPM + день недели + fallback).
